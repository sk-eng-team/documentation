---
title: 'Rate Limitation'
description: 'To ensure fair usage and optimal performance, rate limits are applied to all API endpoints.'
---


| **Tier** | **Requests per minute** | **Description**                 |
| -------- | ----------------------- | ------------------------------- |
| Default  | 60                      | Standard usage tier             |
| Premium  | 300                     | For higher-traffic integrations |

When the limit is reached, the API responds with:

```json
{
  "error": "Rate limit exceeded",
  "message": "Try again in 60 seconds."
}
```

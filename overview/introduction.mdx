## Welcome

Welcome to our API documentation. This guide provides comprehensive information about our API endpoints, authentication methods, and how to integrate with our services.

## Base URL

All API requests should be made to our base URL:

```http
BaseUrl: https://api.yourcompany.com
```

# Authentication

> HiredNow APIs use API Key-based authentication.

<Steps>
  <Step title="First Step: Obtain an API Key">
    Contact your Athena project administrator/ HiredNow support to obtain an API key for your environment (staging(dev) or production).
  </Step>

  <Step title="Second Step: Include API Key in Headers">
    Include your API key in the request header as shown below:

    <Tabs>
      <Tab title="HTTP">
        ```http  theme={null}
        GET /campaigns HTTP/1.1
        Host: api.hirednow.ai
        Authorization: Bearer <YOUR_API_KEY>
        Content-Type: application/json
        ```
      </Tab>

      <Tab title="JSON">
        ```json  theme={null}
        {
          "method": "GET",
          "url": "https://api.hirednow.ai/v1/campaigns",
          "headers": {
            "Authorization": "Bearer <YOUR_API_KEY>",
            "Content-Type": "application/json"
          }
        }
        ```
      </Tab>
    </Tabs>

  </Step>
</Steps>

## Welcome

This guide provides a comprehensive description of the API, including authentication methods, available endpoints, request and response formats, and recommended integration practices. It is intended to serve as a reference for developers to ensure reliable and efficient integration with the provided services.

## Base URL

All API requests must be directed to the following base URL:

```http
BaseUrl: https://api.external.hirednow.ai
```

# Authentication

> HiredNow APIs use API Key-based authentication.

<Steps>
  <Step title="First Step: Obtain an API Key">
    Contact your project administrator at HiredNow or support team to obtain an API key for your environment.
  </Step>

  <Step title="Second Step: Include API Key in Headers">
    Include your API key in the request header as shown below:

    <Tabs>
      <Tab title="HTTP">
        ```http  theme={null}
        GET /campaigns HTTP/1.1
        Host: api.external.hirednow.ai
        Authorization: Bearer <YOUR_API_KEY>
        Content-Type: application/json
        ```
      </Tab>

      <Tab title="JSON">
        ```json  theme={null}
        {
          "method": "GET",
          "url": "https://api.external.hirednow.ai/v1/campaigns",
          "headers": {
            "Authorization": "Bearer <YOUR_API_KEY>",
            "Content-Type": "application/json"
          }
        }
        ```
      </Tab>
    </Tabs>

  </Step>
</Steps>

### GET /campaigns/results

<Note>
  Retrieve campaign performance data such as number of applicants, interviews, and hires for a specific `campaignId`.
</Note>

<Warning>
  **Base URL**: https://api.hirednow.ai/v1
</Warning>

#### Request Endpoint:

```http
GET /campaigns/results?campaignId={campaignId}
```

#### Query Parameters:

| Parameter    | Type     | Required | Description                   |
| ------------ | -------- | -------- | ----------------------------- |
| `campaignId` | `string` | ‚úÖ        | The unique ID of the campaign |

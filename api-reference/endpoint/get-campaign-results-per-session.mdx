### GET /campaigns/results-per-session

<Note>
  Retrieve granular session-based performance data for a specific `campaignId`, useful for tracking trends across recruitment sessions.
</Note>

<Warning>
  **Base URL**: https://api.hirednow.ai/v1
</Warning>

#### Request Endpoint:

```http
GET /campaigns/results-per-session?campaignId={campaignId}
```

#### Query Parameters:

| Parameter    | Type     | Required | Description                   |
| ------------ | -------- | -------- | ----------------------------- |
| `campaignId` | `string` | ‚úÖ        | The unique ID of the campaign |

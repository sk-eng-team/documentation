### GET /campaigns

<Note>
  Retrieve all campaigns associated with a specific `companyId`.
</Note>

<Warning>
  **Base URL**: https://api.hirednow.ai/v1
</Warning>

#### Request Endpoint:

```http
GET /campaigns?companyId={companyId}
```

#### Query Parameters:

| Parameter   | Type     | Required | Description                  |
| ----------- | -------- | -------- | ---------------------------- |
| `companyId` | `string` | ‚úÖ        | The unique ID of the company |
